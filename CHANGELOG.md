## [0.2.0] - 2025-12-07
### 🔧 Ajustes técnicos
- Ajustado `eslint` para a versão **8.57.0** para manter compatibilidade com `eslint-config-next@14`.
- Limpeza de `node_modules` e recriação do ambiente via `pnpm install`.
- Padronização total dos scripts do projeto para **pnpm**.

### 📘 Documentação
- Atualização completa do README com:
  - 💠 Stack técnica atualizada (Next 14, React 18, Tailwind 3.4 etc.)
  - 💠 Badges de status e tech stack
  - 💠 Instruções claras de setup, build e desenvolvimento usando pnpm
  - 💠 Melhoria das seções de SEO e boas práticas
  - 💠 Inclusão de seção opcional de CI/CD usando GitHub Actions
  - 💠 Atualização dos comandos de processamento das galerias
  - 💠 Correções gerais de estrutura e organização

### 🔁 Adicionado
- 🧩 **Fluxo visual do projeto** usando Mermaid, descrevendo:
  - Figma → Exportação → Processamento → Dev → Build → Deploy
